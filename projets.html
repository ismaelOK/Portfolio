Stage Année 2 · Application mobile · Flutter & Firebase

Find — Découverte d'activités en temps réel

Application mobile multiplateforme (Android & iOS) développée avec un camarade durant le dernier stage. Find permet à ses utilisateurs de découvrir des activités organisées près d'eux, d'y participer, de créer les leurs et de discuter avec les autres participants — le tout centré sur les rencontres en personne.

L'application utilise la géolocalisation pour afficher les activités à proximité, avec des filtres avancés (type, distance, date, capacité). Les utilisateurs disposent d'un profil, d'un historique de leurs participations, d'un système de chat par activité et d'un mécanisme de signalement.

Le backend repose entièrement sur Firebase : Firestore pour les données, Firebase Auth pour l'authentification, Firebase Storage pour les médias, et Cloud Functions v2 pour les tâches planifiées (expiration automatique des activités, recalcul des scores de popularité).

Flutter / Dart Firebase Firestore Cloud Functions Géolocalisation flutter_bloc / Cubit Stage · En binôme

Architecture & fonctionnalités

  • Pattern MVVM strict : views → viewmodels → services → models
  • Gestion d'état par Cubit (flutter_bloc)
  • Navigation déclarative via GoRouter
  • Injection de dépendances via GetIt
  • 11 services Firebase encapsulés dans la couche services/
  • Requêtes géospatiales Firestore avec GeoFlutterFire Plus
  • Cloud Function d'expiration planifiée (toutes les 5 min)
  • Tests unitaires et d'intégration

Stage Année 2 · API REST · Collaboration Armée française

API « Aux Claviers Citoyens » — Plateforme e-sport

Dans le cadre de mon second stage, j'ai participé au développement d'une API REST dédiée à la gestion de compétitions e-sport, en collaboration avec la Défense française.

La plateforme permet aux utilisateurs de créer et gérer des comptes joueurs, de former des équipes, de s'inscrire à des tournois et de suivre les résultats de compétitions. L'API expose des endpoints consommés par une interface front-end.

Ma contribution principale a porté sur le module d'authentification : inscription, connexion sécurisée, gestion des sessions (tokens JWT), et contrôle des droits d'accès selon le rôle de l'utilisateur (joueur, capitaine d'équipe, administrateur).

REST API JWT PHP / Symfony MySQL Authentification Stage · Armée fr.

Fonctionnalités développées

  • Inscription et connexion sécurisée (hash bcrypt)
  • Génération et validation de tokens JWT
  • Gestion des rôles (RBAC) et protection des routes
  • Endpoints CRUD pour joueurs et équipes
  • Documentation de l'API

AP · Outil réseau

FluoxIP — Gestion du patrimoine réseau

FluoxIP est un outil de gestion et de recensement des ressources réseau et informatiques d'une organisation. Il permet de recenser les équipements, adresses IP, et de vérifier les niveaux d'habilitation associés aux services.

Ce projet a été l'occasion de mettre en œuvre les compétences liées à la gestion du patrimoine informatique : identification des ressources numériques, respect des normes et standards, et vérification des conditions de continuité de service.
